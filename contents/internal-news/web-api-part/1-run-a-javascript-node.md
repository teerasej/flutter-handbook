
# ทดสอบสร้างโปรเจค JavaScript และใช้ Node รันโค้ดให้ทำงานบนคอมพิวเตอร์

1. จากในโฟลเดอร์โปรเจคหลักของเรา ให้สร้างโฟลเดอร์ชื่อ **web-api**
2. เปิดโฟลเดอร์ **web-api** ขึ้นมาใน Visaul Studio Code
3. คลิกสร้างไฟล์ใหม่ ตั้งชื่อไฟล์ว่า **app.js**
4. พิมพ์โค้ดด้านล่างลงไปในไฟล์ และบันทึกไฟล์

```js
console.log('hello world')
```

5. เปิด Terminal ของ Visual Studio Code 
6. พิมพ์คำสั่งด้านล่าง และกด enter เพื่อรันคำสั่ง

```js
node app
```

7. ควรจะมีข้อความด้านล่าง แสดงขึ้นมาใน console

```
hello world
```
